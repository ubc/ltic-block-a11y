# LTIC Block A11y

A WordPress plugin for accessibility enhancements for the UBC LTIC website. This plugin allows developers to easily add accessibility enhancements to existing WordPress blocks.

## Overview

This plugin provides a structured way to add accessibility enhancements to existing WordPress blocks.
- A JavaScript file that adds accessibility enhancements to blocks using JavaScript hooks.
- A PHP file that adds accessibility enhancements to blocks using PHP hooks.

## Prerequisites

- WordPress 6.5 or higher
- PHP 8.2 or higher
- Node.js and npm (for development)

## Development Setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Build the plugin:**
   ```bash
   npm run build
   ```

3. **Development mode (with watch):**
   ```bash
   npm start
   ```
